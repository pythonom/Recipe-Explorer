# Smart Recipe Explorer 🍲

## 📌 Project Overview
A React-based web app that allows users to search, view, and manage recipes with a modern UI and optimized performance.

## ✨ Features
- Recipe search using API
- Infinite scroll / pagination
- Bookmark system (localStorage)
- Responsive design
- Dark mode (if added)

## 🛠️ Tech Stack
- React (Vite)
- Redux Toolkit / Context API
- React Router
- Axios
- Tailwind CSS

## ⚡ Performance
- Lazy loading
- Memoization
- Optimized API calls

## 📦 Installation
```bash
npm install
npm run dev
```
